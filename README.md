# Task Management System

Учебный проект по разработке системы управления задачами в рамках практической работы №5.

## Состав модулей

### 1. Модуль аутентификации (auth-module)
- Регистрация пользователя
- Авторизация (login/logout)

### 2. Модуль управления задачами (tasks-module)  
- CRUD-операции для задач
- Фильтрация и поиск задач
- Изменение статуса задач

### 3. Модуль отчетности (reports-module)
- Генерация отчетов по задачам
- Статистика выполнения
- Экспорт данных
